1. Notify objects are COM objects in dynamic-link libraries (DLLs). These DLLs are COM component servers. Each type of network component has a class installer that installs the component and registers the COM class objects that the network component owns. After the main installation phase finishes, the system registers the objects. To register a COM class object, the class installer calls the object's DLL entry-point function.
